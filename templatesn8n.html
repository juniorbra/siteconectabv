Não. Os templates são licenciados para uso pessoal e individual. A revenda ou redistribuição não autorizada é proibida.

Posso cancelar se não gostar?

Sim.
Plano anual: pode cancelar dentro dos 7 primeiros dias com reembolso total.

Como recebo os templates?

Os templates são enviados por e-mail, já com o Chat Trigger configurado. Você precisará ajustar conforme a API de WhatsApp que estiver utilizando. Se preferir, pode integrar com o Conecta Botvance para ter acesso a funcionalidades avançadas prontas.

O suporte é oferecido via grupo exclusivo no WhatsApp. Ele cobre dúvidas sobre o funcionamento dos fluxos, mas não inclui personalizações nem ensino técnico. Para um suporte mais aprofundado, você pode contratar uma mentoria individual.
